# Генератор паролей (Java)

Генератор паролей с проверкой надёжности: методы `generatePassword` и
`checkPassword` в `PasswordGenerator.java` (пакет `code`). Готовое демо —
в `App.java`.

## Как запустить

Из корня репозитория (компиляция, затем запуск):

```bash
javac code/*.java && java -cp . code.App
```

Ожидаемый вывод:

```text
== Генерация паролей ==
буквы и цифры:    5vjehYzZEzZ0
со спецсимволами: X@qzNBCMdbAejoX8

== Проверка надёжности ==
abc        -> Слабый пароль (оценка 1 из 5)
abcdef1234 -> Средний пароль (оценка 3 из 5)
Abcdef123! -> Очень надёжный пароль (оценка 5 из 5)
```
