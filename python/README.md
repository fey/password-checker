# Генератор паролей (Python)

Генератор паролей с проверкой надёжности: функции `generate_password` и
`check_password` в `password_generator.py`. Готовое демо — в `main.py`.

## Как запустить

Из каталога `code`:

```bash
python main.py
```

Ожидаемый вывод:

```text
== Генерация паролей ==
буквы и цифры:    5vjehYzZEzZ0
со спецсимволами: X@qzNBCMdbAejoX8

== Проверка надёжности ==
abc        -> Слабый пароль (оценка 1 из 5)
abcdef1234 -> Средний пароль (оценка 3 из 5)
Abcdef123! -> Очень надёжный пароль (оценка 5 из 5)
```
